# **App Name**: Lunea Arte

## Core Features:

- Profile Information: Display the user's name, a brief bio (artist, musician, writer), and a profile picture.
- Project Showcase: Categorize and display projects in Art, Music, and Literature with title
- Social Media Links: Include links to the user's social media profiles such as LinkedIn.
- Contact Information: Provide a way for visitors to contact the user.
- Background Art Generation: Generate abstract color palette as page background tool.
