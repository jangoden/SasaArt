# **App Name**: Lunea Arte

## Core Features:

- Profile Information: Display the user's name, a brief bio (artist, musician, writer), and a profile picture.
- Project Showcase: Categorize and display projects in Art, Music, and Literature with title
- Social Media Links: Include links to the user's social media profiles such as LinkedIn.
- Contact Information: Provide a way for visitors to contact the user.
- Background Art Generation: Generate abstract color palette as page background tool.

## Style Guidelines:

- Primary color: Soft pink (#F2B8C3) to represent creativity and approachability.
- Background color: Light pink (#F9E7EA), a desaturated tone of the primary.
- Accent color: Pale yellow (#F2E3B8) for highlighting important elements.
- Font pairing: 'Belleza' for headlines and 'Alegreya' for body text
- Use simple, clean icons for social media links and navigation.
- Maintain a balanced layout with the profile information centered and projects showcased around it.
- Subtle animations when hovering over projects or social media links.
